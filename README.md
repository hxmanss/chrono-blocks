# Chrono Blocks

## Game Description
Chrono Blocks is an exciting puzzle-platformer that challenges players to manipulate time and navigate through intricate levels filled with various obstacles and enemies. Experience a unique blend of strategy and skill as you guide your character through time-bending scenarios to solve puzzles and complete levels.

## Features
- **Time Manipulation:** Control time to pause, rewind, or fast-forward your character's actions and the environment.
- **Challenging Levels:** Explore a variety of levels, each designed to test your puzzle-solving abilities and reflexes.
- **Engaging Graphics:** Enjoy vibrant and captivating art that enhances the gameplay experience.
- **Diverse Enemies:** Encounter different types of enemies, each with unique patterns and challenges.
- **Power-Ups:** Collect power-ups that grant temporary abilities or enhancements to aid in your quest.

## Setup Instructions
1. **Clone the Repository:**
   ```bash
   git clone https://github.com/hxmanss/chrono-blocks.git
   ```
2. **Install Dependencies:**
   Make sure to have the required dependencies installed:
   ```bash
   npm install
   ```
3. **Run the Game:**
   Launch the game locally:
   ```bash
   npm start
   ```

## Gameplay Mechanics
- **Movement:** Use the arrow keys or WASD to move your character.
- **Time Control:** Activate time manipulation using the designated keys (e.g., `T` for time control).
- **Interaction:** Interact with objects and enemies using the space bar.
- **Objective:** Collect blocks and reach the endpoint of each level to progress.

Dive into the world of Chrono Blocks and test your skills against the clock!
